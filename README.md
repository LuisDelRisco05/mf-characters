# MF Characters (microfront remoto)

Este microfront muestra una lista de personajes (Rick & Morty) usando la API pública.  
Está pensado para usarse como remote por un `root` que carga remotes vía Module Federation (Vite).

---

## Requisitos

- Node.js >= 18 (recomendado 18+ / 20+)
- npm >= 9 o yarn
- Git

---

## Scripts útiles

```bash
# instalar dependencias
npm install

# desarrollo (servidor Vite en puerto 5000)
npm run dev

# build para producción
npm run build

# preview (sirve la build)
npm run preview

# tests
npm run test

# start (el principal para correr el root)
npm run start
